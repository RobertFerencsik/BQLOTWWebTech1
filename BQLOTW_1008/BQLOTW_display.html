CSS - Display Tulajdonság Feladat

display: none;

Ez a bekezdés bemutató szöveg a display: none; tulajdonságról. Az XY1 osztály elemeire ezt alkalmazva az elemek teljesen eltűnnek az oldalról.

PTI Box 1 - Ez az elem nem jelenik meg az oldalon (display: none).

display: inline;

A display: inline; tulajdonság azt jelenti, hogy az elemek egymás mellett helyezkednek el, függőleges és vízszintes térközt nem vehetnek fel:

PTI Box 1 PTI Box 2 PTI Box 3

display: block;

A display: block; tulajdonság teljes szélességet foglal el és új sorba kezd:

PTI Box 1 - Ez az elem teljes szélességet foglal el (display: block).
PTI Box 2 - Ez az elem szintén teljes szélességet foglal el.
PTI Box 3 - Az utolsó elem is új sorban kezdődik.

display: inline-block;

Az inline-block egyesíti az inline és block tulajdonságokat - egymás mellett helyezkednek el, de méretezhetők:

PTI Box 1 - Inline-block elem
PTI Box 2 - Inline-block elem
PTI Box 3 - Inline-block elem

display: flex;

A display: flex; rugalmas elrendezést biztosít az elemeknek:

PTI Box 2
