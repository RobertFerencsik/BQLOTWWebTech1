CSS - Color és Font Size Feladat

Ez egy piros szöveg, amely bemutatja a color tulajdonság használatát. A szöveg piros színű és könnyen olvasható.
Ez a szöveg nagyobb méretű, 24 pixeles. A font-size tulajdonság nagy értékkel beállítva zöld színnel jelenik meg.
Ez a szöveg kisebb méretű, mindössze 12 pixeles. A font-size tulajdonság kis értékkel beállítva kék színnel jelenik meg.
