LSTM Cell Architecture

An LSTM cell is composed of several key components that work together to maintain long-term memory and selectively update information.

1. Cell State (C_t)

The cell state is the "highway" that runs through the entire LSTM chain. It can carry information relatively unchanged from earlier time steps, allowing the network to remember information over long sequences.

2. Hidden State (h_t)

The hidden state is the output of the LSTM cell at each time step. It contains information from both the current input and previous hidden states, making it useful for predictions.

3. Forget Gate

Determines what information should be discarded from the cell state. It uses a sigmoid function to output values between 0 and 1, where 0 means "completely forget" and 1 means "completely keep".

f_t = σ(W_f · [h_{t-1}, x_t] + b_f)

4. Input Gate

Decides what new information should be stored in the cell state. It consists of two parts: a sigmoid layer that decides which values to update, and a tanh layer that creates new candidate values.

i_t = σ(W_i · [h_{t-1}, x_t] + b_i) C̃_t = tanh(W_C · [h_{t-1}, x_t] + b_C)

5. Output Gate

Determines what parts of the cell state should be output as the hidden state. The cell state is passed through tanh, then multiplied by the output gate values.

o_t = σ(W_o · [h_{t-1}, x_t] + b_o) h_t = o_t * tanh(C_t)

6. Complete Update Process

The cell state is updated by first applying the forget gate, then adding new information through the input gate. This allows selective memory management throughout the sequence.

C_t = f_t * C_{t-1} + i_t * C̃_t

LSTM Components Comparison

Component Function Activation Output Range

Input Gate Decides what to store Sigmoid [0, 1]

Output Gate Decides what to output Sigmoid [0, 1]
